# Contributing to AI-Video

Thank you for your interest in AI-Video!

Contributions of all kinds are welcome.

Whether you would like to fix bugs, improve documentation, add new features, or optimize performance, we appreciate your help.

---

# Development Environment

Python 3.12+

Install dependencies:

```bash
pip install -e .
```

Run tests:

```bash
PYTHONPATH=src pytest
```

---

# Coding Style

Please follow these principles:

- Clear and readable code
- Small, focused commits
- Type hints whenever possible
- Meaningful docstrings
- Keep functions simple

---

# Before Opening a Pull Request

Please make sure:

- All tests pass.
- New features include appropriate tests.
- Documentation is updated.
- Code follows the existing project style.

---

# Project Philosophy

AI-Video values:

- Simplicity
- Extensibility
- Reliability
- Maintainability

When adding new functionality, please try to preserve these principles.

---

# Reporting Issues

Bug reports should include:

- Operating System
- Python version
- Steps to reproduce
- Expected behavior
- Actual behavior

---

# Feature Requests

Feature requests are welcome.

Before proposing major architectural changes, please open an Issue for discussion.

Every contribution helps improve AI-Video.
