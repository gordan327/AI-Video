# Terminology

## Purpose

This document defines the official terminology used throughout the AI-Video
documentation.

Its purpose is to ensure consistent language across all documents.

When multiple expressions exist for the same concept, this document defines the
preferred form.

# General Rules

- Use the preferred term consistently.
- Do not introduce synonyms without justification.
- Expand abbreviations on first use.
- Use consistent capitalization.
- Prefer established terminology over stylistic variation.
